Create a Notion-like app that is intended for open source distribution. It should be able to manage a daily Journal for keeping track of the day’s schedules, projects, tasks, activities, notes, resources (web bookmarks), and diary entries. It should also store all data in GitHub, so that archives are version-controlled, distributed (versions and backups), and portable, extensible, and pluggable into other interfaces and applications. It should be a very capable Markdown editor with the ability to connect to a personal library of resources stored in a git provider, such as GitHub, but agnostic, so it can be adapted to work with BitBucket, etc.—maybe with plugable extensions.

It is essentially an adaptive resource connector (ARC) for personal data that is not at the mercy of some large tech company that owns all your data. It is MVP for true data sovereignty.
